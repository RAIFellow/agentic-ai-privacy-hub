This repository contains code to support a paper on an Agentic AI Privacy Hub
The code contains the following
negotiate_retention.py -> A Python script that emulates the negotiation between two AI agents to set data retention
negotiate_retention_sample_output -> A sample output

user_preferences.py -> A Python script showing how verbose user's privacy preferences can be interpreted and enforced by an AI agent
user_preferences_sample_output.py -> A sample output
